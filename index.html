Creating a Counter with JS Classes

Using a class allows us to define the state (the count) and the behavior (increment/decrement) in one blueprint.

The JavaScript Logic

By using this, we ensure that each instance of the counter tracks its own value without interfering with others. 


class Counter {
    constructor(element) {
        this.count = 0;
        this.display = element.querySelector('.display');

        // Bind events
        element.querySelector('.btn-inc').addEventListener('click', () => this.update(1));
        element.querySelector('.btn-dec').addEventListener('click', () => this.update(-1));
        element.querySelector('.btn-reset').addEventListener('click', () => this.reset());
    }

    update(value) {
        this.count += value;
        this.render();
    }

    reset() {
        this.count = 0;
        this.render();
    }

    render() {
        this.display.textContent = this.count;
    }
}

// Initialization
const myCounter = new Counter(document.getElementById('counter1'));
